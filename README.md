# AutoCon 5: Workshop C1: Testing and Validation with Pytest and APIs

This repository contains examples and workbooks used in the AutoCon 5 workshop:
> **AutoCon 5: Workshop C1**
> **Testing and Validation with Pytest and APIs**

## Session Overview

This session introduces validation techniques for network automation. Attendees implement testing using Pytest and device APIs, validating both the intended configuration and the live operational state of the network.

By the end of the session you will have built a testing framework using Pytest and APIs to validate network state with automated checks and reporting.

## Session Details

| Item | Detail |
|------|--------|
| Level | Intermediate to Upper Intermediate |
| Duration | Half day (~4 hours) |
| Tools / Libraries | Pytest, REST APIs, Python |
| Instructor | Rick Donato, Packet Coders |

## Prerequisites

- Basic Python functions
- Understanding of network state and configuration
- Familiarity with automation concepts
- No prior testing framework experience required

## Workbooks

- Workbook 001: Device APIs with eAPI
- Workbook 002: Reshaping Data with JSONata
- Workbook 003: Testing with Pytest

## Lab Environment

All exercises are completed in a browser-based lab. Each attendee is automatically assigned a dedicated environment with workbooks, sample data, and a live device topology pre-loaded, no local install required.
